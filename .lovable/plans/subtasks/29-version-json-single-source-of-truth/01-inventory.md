## Notes / gotchas

- `standalone-scripts/payment-banner-hider/src/index.ts` has BOTH a
  standalone `VERSION` constant AND the `instruction.ts` field. Consolidate
  both to a single import.
- `standalone-scripts/macro-controller/src/shared-state.ts` exports `VERSION`
  used by other files; refactor consumers to import from a single module.
- `readme.md` has one `v4.301.0` pin repeated many times (Pinned version
  section, install snippets, examples table). The propagator can do a simple
  global replace of the previous version → new version, since no other
  `4.NNN.N` numbers appear in prose.
- No test fixtures or snapshots embed the version literal (grep confirms).
