# Memory: architecture/sdk-namespace-convention
Updated: 2026-03-23

The SDK namespace was renamed from `riseupAsia.projects.<camelCase>` to `RiseupAsiaMacroExt.Projects.<PascalCase>`. Root namespace `RiseupAsiaMacroExt` is hardcoded. Each project has two identifiers: `slug` (hyphen-case, e.g., `marco-dashboard`) for file paths/DB names, and `codeName` (PascalCase, e.g., `MarcoDashboard`) for SDK namespace keys. Generated by `slugify()`, `toCodeName()`, and `toSdkNamespace()` in `src/lib/slug-utils.ts`. Full namespace example: `RiseupAsiaMacroExt.Projects.MarcoDashboard`.
