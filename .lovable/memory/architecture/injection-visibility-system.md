The extension utilizes a centralized injection diagnostics system ('src/background/injection-diagnostics.ts') that persists all pipeline events to the SQLite database. To ensure real-time visibility, diagnostics are mirrored to the active tab's console using nested 'console.groupCollapsed' calls. The primary group identifies the injection status (color-coded: green/success, orange/warn, red/error), while internal sub-groups ('📊 Stage Summary' and '📜 Per-Script Results') separate pipeline timings (e.g., Stage 3/4 resolution) from individual script outcomes. These logs include status icons (✅, ❌, ⏭), per-script durations, and specific method-name prefixes (e.g., '[manifest-seeder::fetchManifest]') for traceability. For all file fetch attempts, the system logs full absolute 'chrome-extension://' URLs.
