# Chrome Extension (Generic)

## Overview

Project-agnostic Chrome MV3 extension spec — the portable blueprint a fresh blind AI can apply to any extension codebase. Mirrors this project's architecture (fundamentals, folder + build, TS + linter, storage layers, design system, error management, auth + tokens, injection + host access) but uses `<NAMESPACE>` placeholders (see `spec/00-glossary.md`) so it can be reused outside this repo. Read together with `spec/01-quickstart-for-blind-ai.md` and `spec/02-non-negotiables.md`.
