# PowerShell Integration

## Overview

PowerShell tooling that supports the extension: configuration schema (`01`), script reference (`02`), integration guide (`03`), error codes (`04`), firewall rules (`05`), PHP known-issues (`06`), and multi-site deployment (`25`). Used by `browser-deploy.ps1` and friends — see `mem://architecture/deployment-diagnostics`. Root readme authoring order (`mem://workflow/root-readme-authoring-order`) places PowerShell install steps **before** macOS/Linux.
